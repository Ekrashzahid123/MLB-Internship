# Python Data Structures Practice

## Overview

This project contains practice tasks completed  
The main focus was learning and implementing basic Python data structures such as:

- Lists
- Tuples
- Sets
- Dictionaries

The tasks were solved using basic programming concepts like loops, conditions, and variables without depending on many built-in functions.

---

## What I Learned Today

During today's practice, I learned:

### Lists
- How to store multiple values in a single variable.
- Finding the largest and second largest number.
- Removing duplicate values manually.
- Reversing a list using loops.
- Comparing two lists to find common elements.

### Tuples
- Understanding immutable data structures.
- Counting elements using loops.
- Converting tuples into lists and lists into tuples.

### Sets
- Understanding unique values.
- Performing union and intersection operations.
- Removing duplicate data.

### Dictionaries
- Creating and accessing records using key-value pairs.
- Storing student information.
- Calculating average marks.
- Counting word frequency in a sentence.

## Challenges Faced

Some challenges I faced during the tasks:

- Understanding how to solve problems without using built-in functions.
- Finding the second largest number using logic.
- Removing duplicate values while keeping the original order.
- Converting between different data structures.
- Handling loops for searching and counting operations.

---

## Solutions Implemented

To solve these challenges:

- Used `for` loops and `while` loops for iterations.
- Used `if-else` conditions for decision making.
- Created temporary lists to store results.
- Compared values manually instead of using functions like:
  - `max()`
  - `reverse()`
  - `count()`
  - `set()`
- Practiced dictionary operations using key-value pairs.
